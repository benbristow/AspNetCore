# BenBristow.AspNetCore

A utility library collection for ASP.NET Core applications, available as separate NuGet packages.

## Packages

- **[BenBristow.AspNetCore.DataValidationException](BenBristow.AspNetCore.DataValidationException)**

Provides a custom exception type for handling data validation errors in ASP.NET Core applications.
